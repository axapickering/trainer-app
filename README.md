# Quickstart:
1. Start up frontend
   - ```cd frontend```
   - ```npm install```
   - ```npm start```
   - Open [http://localhost:3000](http://localhost:3000) to view in browser

2. Start up backend server
   - ```cd backend```
   - ```npm install```
   - ```npm run dev ```
   - Server should be running on ```http://localhost:5000```
